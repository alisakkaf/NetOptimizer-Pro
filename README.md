**NetOptimizer Pro** is the ultimate Windows Bandwidth Control and ZeroLeak script. It is aggressively designed to block background data consumption, disable aggressive Windows telemetry, and stop silent browser auto-updates. Perfect for developers, gamers, and users on metered connections who need **100% control over their network bandwidth**.

---

## 🚀 Key Features

### 🛡️ System Network Control
* **ZeroLeak Protocol:** Forcefully stops background updates (`wuauserv`, `bits`, `UsoSvc`).
* **Telemetry Blocker:** Disables Windows diagnostic tracking and data collection.
* **Bandwidth Throttling:** Automatically enforces "Metered Connection" on WiFi and Ethernet adapters.
* **Safe Restoration:** 1-Click option to safely restore all default Windows services.

### 🌐 Browser Auto-Update Elimination
* **Hard Kill Switch:** Terminates hidden background updater processes in RAM.
* **GPO Injection:** Applies Local Group Policies to permanently block internal updates for **Chrome, Brave, Edge, and Firefox**.
* **Scheduled Tasks Eradication:** Removes hidden update triggers.

---

## ⚠️ Disclaimer & Warning
This is an **aggressive system script**. It directly modifies Windows Services, Scheduled Tasks, and Registry Policies to maximize network performance. 
* Do not use this as a permanent daily driver without understanding the security risks of pausing Windows Updates.
* Always re-enable updates periodically to receive critical security patches.
* **Use at your own risk.** The developer is not responsible for any system instability.

---

## 🛠️ How to Use

1. Download the latest `NetOptimizer_Pro.bat` script.
2. Right-click the file and select **"Run as Administrator"** (The script has an auto-elevation feature, but doing it manually is best practice).
3. Use the dynamic ANSI color menu to choose your execution protocol.
4. Restart your PC if recommended by the script prompt.
