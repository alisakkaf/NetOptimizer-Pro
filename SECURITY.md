# Security Policy

## Supported Versions
Currently, only the latest version of `NetOptimizer Pro` (v2.0 and above) is supported with security updates.

## Reporting a Vulnerability
Since this script heavily modifies Windows Registry and Core Services, security is a top priority.
If you discover a vulnerability or a critical bug that causes system instability, please **DO NOT** open a public issue. 


I will do my best to respond and issue a hotfix within 48 hours.
